# 石門國小資訊科技駕駛艙：答題快打

線上遊玩：https://cagoooo.github.io/smes-it-quiz-battle/

給國小學生使用的資訊素養答題對戰遊戲。直接開啟 `index.html` 即可使用，無需登入或後端。

## 課堂玩法

1. 選擇單人闖關或同桌對戰、資訊主題與角色；共有 6 位原創學生英雄可選。
2. 單人闖關會隨機遭遇防火牆怪、雜訊干擾獸或漏洞幻影三位 CPU 敵人。
3. 每回合選一種技能；題目難度越高，答對造成的效果越強。能量集滿後，可施放每場隨機抽選的角色專屬大絕招。
3. 答錯會立即顯示正確答案與簡短說明，可作為全班討論素材。
4. 答對累積能量後可使用「終極防護」。

題庫已擴充至 45 題以上，涵蓋網路安全、程式思維、數位公民與綜合資訊素養。進度與對戰完全在瀏覽器本機執行，不會蒐集學生資料。
