# 石門國小資訊科技駕駛艙：答題快打

線上遊玩：https://cagoooo.github.io/smes-it-quiz-battle/

給國小學生使用的資訊素養答題對戰遊戲。直接開啟 `index.html` 即可使用，無需登入或後端。

## 課堂玩法

1. 選擇單人闖關或同桌對戰、資訊主題與 8 位原創學生英雄。
2. 單人闖關會隨機遭遇 8 種原創 CPU 敵人。
3. 每位角色都有三種專屬一般技能與一種每場隨機抽選的專屬大絕招；技能的名稱、傷害、題型與能量累積各不相同。
4. 答錯會立即顯示正確答案與簡短說明，可作為全班討論素材。
5. 大絕招會進行 6–9 段逐段命中的連招演出，每段即時扣除對手專注力。

題庫涵蓋網路安全、程式思維、數位公民與綜合資訊素養，共 117 題。進度與對戰完全在瀏覽器本機執行，不會蒐集學生資料。

## 課前檢查

上課前可在專案根目錄執行：`./scripts/preflight-check.ps1`。它會檢查 JavaScript 語法、P0 核心互動回歸與 PWA 版本一致性。

## Suno 音樂機制

已支援大廳、對戰、終極技能、勝利與重新挑戰五個音樂場景。正式 Suno 音檔與授權／SHA-256 稽核方式請見 [`assets/audio/README.md`](assets/audio/README.md)；還未放入審核 MP3 時，遊戲會自動使用內建合成節奏作為備援。
